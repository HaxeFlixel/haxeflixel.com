Download

Haxeflixel is available to use on Linux, Mac or Windows. You will need to have Haxe 3 and OpenFL installed and working.

If you need help setting this up, please see the Getting Started article.

We recomend you use Haxelib with the following command:

haxelib install flixel

To download the latest stable code use master branch, or this link to the zip:

Download Zip

We also recommend users inspect the demos to see Flixel features in use.

The documentation and forums provide the main source of community created docs, support and resources.

For advanced developers, the source code is available and open for contributions on GitHub, bleeding edge code is on the dev branch.
