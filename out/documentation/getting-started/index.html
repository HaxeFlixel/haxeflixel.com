Getting Started

Edit

Welcome to HaxeFlixel, here we will show you how to setup your system to start making games with Mac, Linux or Windows.

Getting started requires three steps:

  1. Install OpenFL and Haxe
  2. Install HaxeFlixel
  3. HaxeFlixel Hello World

Time to complete this installation is close to 20 minutes. (depending on the speed of your Internet connection for downloading the components)

If you require some help:

  • Ask a question in the community forums;
  • Join the irc channel #haxeflixel on freenode.

This guide is an open and community effort if you think you can help us improve it, please submit a pull request on GitHub. Doing so is made easier from the direct github edit link on the top right hand side of each page.
