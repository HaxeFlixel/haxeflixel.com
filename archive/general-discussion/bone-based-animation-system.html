Bone based animation system

crazysam, 10 months 4 weeks ago

Spriter is a program to create bone based animations which are great for mobile games since you can create a wide range of animations with relatively small texture memory usage. Checkout a video of what it can do.

 

I've been following the forums and someone finally created a stable AS3 implementation of the Spriter file format: http://www.brashmonkey.com/forum/viewtopic.php?f=3&t=6431

 

The AS3 implementation of the plugin was developed for Starling. And uses the normal display chain by inheriting Sprite. Check out the code here.

 

Integrating this is probably more of a long term project (since the AS3 implementation is v0.01), so I'm just gathering some ideas and suggestions. Should characters animated this way be built out of FlxSprites? Or maybe this would be a good use for the NestedSprite class?
