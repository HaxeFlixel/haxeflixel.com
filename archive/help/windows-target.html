HalfDayDeemo
4 months 1 week ago

If this is your first time you have been bitten by an order of execution issue then it won't be your last. :D

You should be fine if you follow some simple steps. One dead easy way is to have a state like 'Pre-Play' or 'Loading' because that might actually scale better down the line when you start to load more. So it would go;

MenuState -> Start Game -> Move to LoadingState

LoadingState -> Populate the registry. -> Move to PlayState when loaded.

PlayState -> Use all the stuff you just created.

Initially the loading state will just get skimmed over real quick, *but* it gives you the option later to add a 'Loading...' animation in future if you start to require longer setup times, i.e. loading lots of graphics.
